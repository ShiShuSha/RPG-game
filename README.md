# RPG Game

Простой RPG-проект на JavaScript с персонажами, оружием и системой боёв.

## Установка

1. Откройте терминал в папке проекта.
2. Установите зависимости:

```bash
npm install
```

## Запуск проекта

Для запуска локального сервера:

```bash
npm start
```

## Сборка

Для сборки скриптов в папку `dist`:

```bash
npm run build
```

## Тесты

Для запуска тестов Jest и проверки покрытия:

```bash
npm test
```
